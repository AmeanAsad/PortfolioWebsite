Syndrome Error Decoding

Github Link

I developed a class that is able to develop any (n, k) linear code including by generating a generator matrix, parity check matrix, and a syndrome decoding table. I used that class to develop an ASCII code class in order to correct errors in the ascii representation of alphabet letters.

I developed a stochastic simulation in order to test the accuracy of different error correcting codes. This was done by writing a random text generator. I used a 5000 word data set an input to the generator. I used the output from the text generator to feed it into a function that parses the text and transforms it into a vector of binary representations of each character based on it's ASCII representation. I then used a function that randomly distorts bits in each character. From there I was able to apply the linear code to try and retrieve the original text. My simulation focused on measuring the time required for the operation and also the amount of errors that where corrected.

The result of simulating the decoder over a noisy channel with a 12.5% error rate. I plotted the word count verses the time multiplied by error percentage in order to factor in the time the linear code takes to recover the message as a performance parameter.

Forest Wildfire Mitigation System

Github Link

For one of our design projects in Engineering, we were tasked with implementing multi-agent dynamics to solve a problem. Our group implemented multi agent dynamics on autonomous, decentralized drone agents to optimize aerial wildfire mitigation.

One of the issues we ran into is simulating our algorithm over an actual fire spread. I decided to take the lead on solving this issue. I decided to stochastically model the fire spread using Python, which would allow us to test our algorithm over various fires that are randomly generated. I studied some cellular automaton concepts and modelled the stochastic spread of each cell based on its Moore Neighborhood. I also incorporated some parameters to impact the spread, such as a wind vector field and moisture levels of the terrain.

I implemented Lloyd’s deployment algorithm in Matlab along with Voronoi tessellations to determine agent movement based on various fire parameters and communication with other agents. Then I wrote a script in Matlab to connect the fire simulation data from Python as an input to the algorithm, and the simulation averaged mitigating 60% of the developed fires using just aerial drones.

This is an image of the simulated fire spread. The gradient in the image displays the overall fire intensity in that region.

A fire simulation where 6 drones(blue dots) where used to mitigate the fire. The voronoi tessellations are shown where the centroids are represented by the red dots.
